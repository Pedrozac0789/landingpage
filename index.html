Sobre o Projeto

Este projeto tem como objetivo criar uma aplicação completa, unindo front-end, back-end e banco de dados em uma arquitetura moderna. No front-end, usei HTML , CSS e JavaScript para desenvolver interfaces responsivas e fáceis de usar. No back-end, construímos uma API RESTful com Node.js, responsável por processar requisições e aplicar regras de negócio. Para armazenar os dados, adotei o MySQL, com modelagem e consultas organizadas pelo MySQL Workbench. A integração dessas tecnologias resultou em um sistema escalável, modular e de fácil manutenção, garantindo boa performance e uma experiência fluida para o usuário.

Funcionalidades

  • Cadastro de compromissos com título e horários
  • Visualização semanal organizada por dias
  • Edição e exclusão de compromissos
  • Calendario interativo
